### ✨ 주요 기능

- 반응형 Swiper: 작은 화면 (< 767px)에서는 슬라이더를 활성화하고, 큰 화면에서는 슬라이더를 비활성화
- 슬라이드 수 설정: 작은 화면에서 슬라이드가 1.5개씩 보이도록 설정
- 리사이즈 처리: 화면 크기에 맞춰 슬라이더 초기화 및 파괴
